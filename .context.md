# Shared Types Library

## Stack
TypeScript only — no runtime, no dependencies

## Purpose
Shared types, constants, and utilities used by all other packages.

## Key Files
| File | Purpose |
|---|---|
| `src/index.ts` | Core types: `IDEEvent`, `Command`, `Message`, `FileEntry`, `GitStatusData`, WebSocket protocol types |
| `src/auth.ts` | Auth types: `UserProfile`, `SubscriptionInfo`, `RemoteSession`, `UsageStats`, `DashboardPreferences` |
| `src/pricing.ts` | Pricing plans: `individual`, `company`, `custom` with crypto payment types |

## Message Protocol
All WebSocket messages are JSON. Three types:
- `handshake` — role + token, sent on connect
- `event` — IDE events from extension (file_opened, build_complete, etc.)
- `command` — actions from mobile (run_command, open_file, git_status, etc.)
- `response` — success/error replies with optional data

## IDEEvent Types
- `file_opened` — active editor changed
- `cursor_moved` — cursor position update
- `terminal_output` — stdout/stderr chunk
- `git_status` — branch + changes list
- `task_started` / `task_completed` — task lifecycle
- `build_complete` — build/test task result (M1 feature)
